Eastgate Character Manager. Alpha Version!

If you're new, please go to the login page, register, go to the character sheet page and make a new character!

Characters will now update if the database changes. So you can save without fear of having to remake your character. If you encounter any problems saving information, try refreshing and reentering the data. If the problems persist, please contact me.
